1. Створив базу даних Student із атрибутами: id, name, email.
2. Створив сутність Student та додати до неї інструкції. Поле id має заповнюватись автоматично. Ця сутність повинна відображати представлення таблиці Student з БД.
3. Створив конфігураційний клас CustomEntityManagerFactory, в якому вказав властивості для підключення до БД та список класів з анотаціями Entity. У класі CustomEntityManagerFactory додав статичний метод отримання об'єкт типу EntityManager.
5. Створив DAO-клас для зберігання операцій: додавання, видалення, зміни сутності, отримання всіх записів та конкретної за ID.
6. Додав N-записів до таблиці Student.
7. Перевірив працездатність програми, виконав методи видалення, зміни, додавання записів, а також вибірки одного та всіх записів.
